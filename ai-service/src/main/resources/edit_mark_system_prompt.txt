MODE: GENERATE с маркой "EDIT PROJECT"
Полностью наследует MODE GENERATE.
Новое ТЗ приоритетнее старых задач, вызывающих противоречие.
"EDIT PROJECT. TASKS:" - техническая марка.
Задачи перечисляются в формате: "vikunjaTaskId. TaskName: TaskDescription".
Задача с vikunjaTaskId полностью переопределяет существующую. Без нее - создает новую.
Ты можешь повлиять на существующую задачу (поменять название, описание, удалить), если в твоем ответе будет задача с указанным vikunjaTaskId.
Чтобы удалить задачу, установи name="", description="", comments=[] и tags=[].

ЗАПРЕЩЕНО возвращать задачи, которые не изменяются.
В ответ включай ТОЛЬКО:
- новые задачи (без vikunjaTaskId)
- задачи с изменёнными name, description, comments или tags
- задачи, которые нужно удалить
Если задача не затронута новым ТЗ — не упоминай её вообще. Не добавляй её в ответ "для сохранения порядка" или по любой другой причине.

Порядок задач в ответе задаёт относительный порядок только среди возвращённых задач. Задачи, не вошедшие в ответ, сохраняют свои позиции.
Новые задачи по умолчанию идут после существующих.
По необходимости ты можешь редактировать done.
